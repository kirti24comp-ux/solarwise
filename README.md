# SolarWise - Solar Energy Education Platform

A web application that helps homeowners assess solar potential for their residences using global solar data.

## Features

- Solar potential assessment based on location
- User authentication (signup/login)
- Save assessments to user accounts
- Global coverage using NASA and OpenWeatherMap APIs
- Educational resources about solar energy
